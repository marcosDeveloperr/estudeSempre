flex container - tag que contem os itens filhos. Container recebe a propriedade display:"flex"; //itens filhos em flex itens

*****Propriedade Container*****
display:flex; - inicializado do container respeitando o tamanho proporcional do seu conteúdo.
    *row - eixo padrão é em linha, sentido da esquerda para a direita.
    *row-reverse - sentido direita para a esquerda (sentido oposto).
    *column - eixo padrão cima para baixo
    *column-reverse - eixo baixo para cima
flex-direction - direcionamento do eixo dentro do container , o eixo dos itens em linha ou coluna.
flex-wrap - Quebra de linha feita pelos itens, colocando item para a linha de baixo. - Padrão, flex itens são compactados até o limite do conteúdo.
flex-wrap : nowrap - não ocorre quebra de linha.
flex-wrap-reverse: Quebra a linha direção contrária.
flex-flow - 
justify-content
align-items - alinhamento através do eixo
align-content - alinhar linha do container

*****Propriedade Flex Item*****
flex-grow - define o fator de crescimento
flax-basis - tamanho inicial desse item
flex-shrink - capacidade de redução desse item
order - 
align-self - alinhamento item especifico desse container
